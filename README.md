This repository contains my hands-on learning and practical implementation of Terraform for Infrastructure as Code (IaC). Through this project, I explored how to automate cloud infrastructure deployment, manage resources efficiently, and follow DevOps best practices using Terraform.

- Infrastructure as Code (IaC)

- Terraform State Management
- Modules in Terraform
- Provisioning Cloud Infrastructure
- Terraform Commands & Workflow
- Remote Backend Configuration
- Best Practices for Reusable Infrastructure
